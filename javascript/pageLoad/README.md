 问题一：从用户输入URL到页面最后呈现，有哪些过程？
 考察：网络原理、浏览器加载css js 解析html过程
 大致到过程如下：
 1、DNS解析
 2、TCP连接
 3、发送HTTP请求
 4、服务器处理请求并返回http报文
 5、浏览器解析渲染页面
 6、连接结束
